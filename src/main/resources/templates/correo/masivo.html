📧 Envío Masivo de Correos

Envía correos a múltiples destinatarios de forma rápida y segura

💡 Información: Esta herramienta está disponible solo para administradores. Los correos se envían de forma asíncrona por lotes para mejor rendimiento.
✉️ Contenido del Correo
Máximo 200 caracteres
Puedes usar HTML básico (p, br, strong, em, a, etc.). Si usas plantilla profesional, este contenido irá en el cuerpo.
🎨 Apariencia del Correo
Activar esta opción envolverá tu contenido en una plantilla HTML profesional con logo, colores corporativos y opciones de desuscripción.
Se usa para generar enlaces de desuscripción y referencias a recursos estáticos en la plantilla profesional.
👥 Destinatarios
Separa múltiples correos con saltos de línea, comas o puntos y comas.
📁 Arrastra un archivo aquí o haz clic para seleccionar
El archivo debe contener un correo por línea. Formatos soportados: .csv, .txt
⚠️ Nota: Se incluirán automáticamente todos los correos de usuarios registrados en el sistema, más los que proporciones aquí.
